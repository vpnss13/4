Shadowsocks软件介绍
SS，全称为 Shadowsocks（下文简称为SS），是一种基于 Socks5 代理方式的网络数据加密传输包的工具，可以使用其于科学上网 也就是翻墙 ，是目前最稳定，最常用的科学上网工具之一，全平台支持

SSR，全称为 ShadowsocksR（下文简称为SSR） 来历：

ShadowsocksR是breakwa11发起的Shadowsocks分支，在Shadowsocks的基础上增加了一些数据混淆方式，称修复了部分安全问题并可以提高QoS优先级。后来贡献者Librehat也为Shadowsocks补上了一些此类特性，甚至增加了类似Tor的可插拔传输层功能。

ShadowsocksR开始时曾有过违反GPL、发放二进制时不发放源码的争议，不过后来ShadowsocksR项目由breakwa11转为了与Shadowsocks相同的GPL、Apache许可证、MIT许可证等多重自由软件许可协议。

2017年7月19日，ShadowsocksR作者breakwa11在Telegram频道ShadowsocksR news里转发了深圳市启用SS协议检测并被大量用户转发，引发恐慌。7月24日，breakwa11发布了闭源的SS被动检测程序，引发争议。 2017年7月27日，breakwa11遭到自称 “ESU.TV” 的不明身份人士人身攻击，对方宣称如果不停止开发并阻止用户讨论此事件将发布更多包含个人隐私的资料，随后breakwa11表示遭到对方人肉搜索并公开个人资料的是无关人士，为了防止对方继续伤害无关人士，breakwa11将删除GitHub上的所有代码、解散相关交流群组，停止ShadowsocksR项目。但项目已被多人fork。并有人在其基础上继续发布新的版本，例如SSRR

Shadowrocket：上传、VPS配置与加速指南
Shadowrocket是一款专为iOS设备设计的网络代理工具，提供用户方便快捷的翻墙解决方案。随着互联网对自由的追求，Shadowrocket的使用越来越普遍，尤其是在需要访问国外网站的情况下。本文将讨论Shadowrocket的上传功能、VPS配置、加速技巧以及在台湾的使用情况，并简要介绍MacBook上的使用与开发。

1. Shadowrocket的上传功能
Shadowrocket允许用户上传自定义的配置文件，包括各种代理节点的设置和规则。用户可以通过URL或直接文件上传的方式，导入各种脚本和配置，从而轻松管理和切换不同的网络环境。通过这些配置，用户可以快速访问需要的内容，无需手动逐个添加节点。

上传步骤：

打开Shadowrocket应用。
在“配置”页面选择“添加配置”。
选择“导入”选项，可以通过URL或直接上传文件。
确认并保存设置，完成后即可使用新的配置。
2. VPS配置与使用
使用VPS（虚拟私人服务器）可以提高Shadowrocket的速度和稳定性。通过自建VPS，用户可以更好地控制代理节点，保证网络访问的隐私与安全。

VPS配置步骤：

选择服务商：选择适合的VPS提供商，如AWS、Vultr或Linode。
安装代理软件：常见的有Shadowsocks或V2Ray，可以根据需要进行选择。
配置节点信息：设置服务器的IP地址、端口、加密方式等信息。
在Shadowrocket中添加节点：输入VPS的连接信息，保存后即可使用。
3. 加速技巧
为提高Shadowrocket的连接速度，用户可以尝试以下几种方法：

选择优质节点：通过对比不同节点的速度和延迟，选择最快的节点进行连接。
使用加密协议：选择更高效的加密协议（如AEAD），可以提升连接速度。
设置DNS解析：使用快速的DNS服务（如Google DNS或Cloudflare DNS）可以提高域名解析速度，减少连接时间。
4. 台湾的使用情况
在台湾，Shadowrocket同样被广泛使用。由于地理位置的优势，连接到邻近的服务器通常会有较低的延迟和更快的速度。用户可以选择台湾本地的VPS节点，以获得更稳定的连接体验。此外，台湾的网络环境相对开放，使用Shadowrocket的用户也会相对较多。

5. MacBook上的使用
虽然Shadowrocket主要是为iOS设计，但在MacBook上也有类似的工具可供使用，如ShadowsocksX-NG或V2RayU。用户可以通过这些应用实现与Shadowrocket相似的功能，方便地翻墙和管理网络连接。

6. Shadowrocket的开发与更新
Shadowrocket的开发者定期发布更新，优化用户体验并修复bug。用户可以关注开发者的官方渠道，了解最新的功能与修复内容。开发者社区也提供了大量的脚本和配置，方便用户自定义自己的使用体验。
